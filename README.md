# Magic series

Search magic series using constraints programming

----------------------

__Def__ A series $S=(S_0, S_1, ..., S_n)$ is magic if $S_i$ represents the number of occurences of $i$ in $S$


|   | 0 | 1 | 2 | 3 | 4 |
|---|---|---|---|---|---|
|Occurences| ? | ? | ? | ? | ? |

**For example**:

|   | 0 | 1 | 2 | 3 | 4 |
|---|---|---|---|---|---|
|Occurences| 2 | 1 | 2 | 0 | 0 |

 - 2 occurences of 0
 - 1 occurence  of 1
 - 2 occurences of 2
 - 0 occurences of 3
 - 0 occurences of 4
